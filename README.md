# 今晚去哪｜約會行程推薦

以靜態網頁整理台北一帶**咖啡館與適合碰面的口袋店**，幫你和朋友（或約會對象）快速挑地點、排一條當日動線。介面為深色、閱讀優先，手機與桌機都可使用。

## 這個專案在做什麼

- **首頁（`index.html`）**  
  瀏覽策展過的店家與區域線索，當成「今晚要去哪」的靈感牆。

- **行程規劃（`planner.html`）**  
  依你輸入的條件與時段，從名單裡挑出較合適的造訪順序；營業時間若未建檔，頁面會提醒以 Google 地圖等官方資訊為準。

- **資料（`data/places.js`）**  
  店名、區域、標籤、地圖搜尋字串、人潮與公休備註等，可依同一套 `id` 與首頁口袋清單對齊擴充。

- **選題／蒐集輔助（`scripts/`）**  
  以 Python 腳本搭配關鍵字設定，從網路上整理美學／空間相關文章線索，輸出到 `output/`，方便人工篩選後再補進前端資料（實際營業與地址請一律向店家或地圖查證）。

## 本機預覽

這是純前端專案，無需建置。在專案根目錄啟動任一靜態伺服器即可，例如：

```bash
python3 -m http.server 8080
```

瀏覽器開啟 `http://localhost:8080/index.html`。

（`planner.html` 需透過 http 開啟，以便正確載入 `data/places.js`。）

## 腳本依賴

```bash
pip install -r requirements.txt
```

詳細用途見 `scripts/crawl_aesthetic_articles.py` 與 `aesthetic_keywords.json`。
